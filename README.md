# Планчик — Telegram Mini App

Mini App для бота @astro_byrbot (astrology).

**Что внутри:**
- Таро-совет дня (78 карт, детерминировано по telegramId+date)
- Биоритмы (физ 23д / эмо 28д / инт 33д)
- Вечерний чек-ин (5 эмодзи + заметка)
- Профиль и история

**Стек:** vanilla HTML+CSS+JS, без бэкенда, без зависимостей (кроме Telegram WebApp SDK с CDN).

**Деплой:** GitHub Pages, URL подставляется в `WEBAPP_URL` env бота.

**Автор:** @Byrja (Планчик — Astro Natal Bot)
